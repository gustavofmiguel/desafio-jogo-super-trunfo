#  Super Trunfo de Cidades em C

Projeto desenvolvido como **desafio** da faculdade para criar as cartas do jogo Super Trunfo de Cidades usando a linguagem C.

## ✏️ Descrição do projeto

Este programa em C realiza o cadastro, exibição de cartas, e compara atributos de cada carta do jogo Super Trunfo de cidades, utilizando apenas conceitos básicos de programação estruturada. <br>
<br>
Foi desenvolvido conforme os parâmetros solicitados
Adicionando novos apredizados confome seguiamos nas aulas
a implementação segue um fluxo do basico ao avançado usando o mesmo código
<br>

Com foco em aperfeiçoar os conhecimentos em lógica
- Variáveis e formatos I/O
- Operadores aritméticos
- Atribuições compostas  
- Incremento/decremento
- Operadores relacionais 
- Estruturas Condicionais  `if` / `else` / `else if`
- Operadores lógicos (`&&`, `||`, `!`)
- Laços `while` / `do-while` / `for`
- switch-case
- break / continue
